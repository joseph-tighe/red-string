# Red string

Red string is a chrome extension that generates a MLA 8 citation for the current page.

## Using the extension
The extentions for both Firefox and Google Chrome have to go through a review process and may not be updated to the latest version the latest version right now is 26.7.8

[Firefox install for 26.7.3](https://addons.mozilla.org/en-US/firefox/addon/red-string-reaserch/)

[Google Chrome Install 26.7.3](https://chromewebstore.google.com/detail/red-string/oejgiicopolenjojlibfjiachjbmaimp)

> NOTE
> 
> 26.7.3 does not have support for several citation formats it only has suppport for MLA 8
## How to use
1. Download the extention
2. On the top of your tab bar press the extentions icon
3. Press the red string icon <img width="16" height="16" alt="icon-128" src="https://github.com/user-attachments/assets/5ef5cfe1-fd6e-4af7-8064-b354dab244be" />
4. This should open a pop-up with the current citation
## Features
- Generates citation for the current page
- Copies citation to clipboard
- View all citations
- Supported citation formats include Chicago, ADA, and MLA 8
## Find previous citations
1. Open the popup
2. Press the link at the bottom of the popup
## Citation formats
1. Open the popup
2. Press the settings gear in the top right corner
3. Select the button that coresponds to the citation format you want
4. All citations across the extention will now be in that citation format including previously saved citations
## How it works
It mostly uses metadata as well as a few elements in the html, the URL, and the page title to find all the information required for the citation.
> Note:
> 
> Because we use metadata, and there is no good standered for what that metadata should look like please keep in mind for some sites the algorithim may retrive the wrong metadata or not format it correctly please feel free to make an issue on github for any broken sites.
